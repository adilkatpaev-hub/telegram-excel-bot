# Telegram-бот для Excel-отчета

Бот принимает данные в Telegram и добавляет их в Excel-файл `report.xlsx`.

## Колонки отчета

- Фамилия
- Дата
- Наименование устройства или адреса
- Проделанная работа

## Подготовка

1. Установите Python 3.11 или новее.
2. Создайте Telegram-бота через `@BotFather`.
3. Скопируйте токен бота.
4. Создайте файл `.env` рядом с `bot.py`:

```env
BOT_TOKEN=ваш_токен_бота
```

## Установка зависимостей

Можно просто запустить файл:

```text
install.bat
```

Или выполнить команды вручную:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

Если команда `python` не работает, установите Python с официального сайта и включите пункт `Add python.exe to PATH`.

## Запуск

Можно просто запустить файл:

```text
start_bot.bat
```

Или выполнить команду вручную:

```powershell
python bot.py
```

После запуска откройте бота в Telegram и отправьте:

```text
/start
```

Бот по очереди спросит фамилию, дату, устройство/адрес и проделанную работу. После последнего ответа запись появится в файле `report.xlsx`.
